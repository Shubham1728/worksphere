# WorkSphere

WorkSphere is an enterprise-grade project management and collaboration platform designed for modern software teams.

It helps organizations manage projects, sprints, tasks, bugs, QA workflows, releases, and team collaboration through a secure, scalable architecture.

## Features

- Authentication & Authorization
- Project Management
- Sprint Planning
- Kanban Board
- Bug Tracking
- QA Workflow
- Dashboard & Reports
- Notifications
- Audit Logs

## Tech Stack

Backend:
Java 21
Spring Boot
PostgreSQL

Frontend:
React
TypeScript

Infrastructure:
Docker
